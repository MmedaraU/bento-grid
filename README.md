### Fun things I did and What I learned
* Self-hosted a font
* Got to use `font-weight` of `600`, medium-bold
* Got to learn a better way to implement responsive typography
  Using this:
  ```css
    html {
    font-size: 100%;
    }
  ```
  Instead of just this:
    ```css
    body {
    font-size: 100%;
    }
  ```
